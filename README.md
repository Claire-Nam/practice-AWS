## 주차별 학습 목표
<h3>1주차: </h3>
<p>
  1. EC2를 생성하고, VPC, Subnet, Route Table, Security Group을 설정하면서 각 구성요소가 어떠한 일을 하는지를 이해한다. </br>
  2. 이를 바탕으로 NAT Instace를 생성하고, NAT Gateway와의 차이를 이해한다.
</p>
<p>2주차: </br> 
  1. EKS 클러스터와 Worker Node를 생성하고, Nginx 서비스를 배포한다. </br>
  2. EKS 클러스터가 무엇인지 마스터 노드와 워커 노드의 차이를 이해한다.
</p>
<p>3주차: </br> 
  1. EFK 환경을 구성하고 각 항목을 배포 및 연결하여 최종적으로 로그를 수집하고 시각화한다. </br>
  2. Elastic Search, Kibana, Fluentbit 각각의 역할을 이해하고, 사용법을 숙지한다.
</p>
<p>4주차: </br> 
  1. CloudWatch를 이용하여 실시간 장애 모니터링 라인을 구축한다. </br>
  2. Stress를 이용하여 실제 CPU 등 서버에 부하를 주고, 슬랙을 통해 장애 모니터링 알람이 오는지를 실습한다.
</p>
